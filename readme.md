# Zerodha Clone

A full-stack MERN application inspired by Zerodha, designed to simulate stock trading operations with a modern and responsive user interface.

## 🚀 Features

* Buy and Sell Stocks
* Portfolio & Holdings Management
* Orders Management
* Interactive Dashboard
* Real-time Data Visualization
* Responsive Design

## 🛠️ Tech Stack 

### Frontend

* React.js
* Bootstrap
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

### Dashboard

* React.js
* Chart.js

## 📂 Project Structure

```text
frontend/
backend/
dashboard/
```

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Dashboard

```bash
cd dashboard
npm install
npm start
```
